Operadores Relacionais: símbolos especiais que realizam comparações entre determinados operandos
- similaridade(ex:==/!=) -tamanho(ex:>=/<=)
- não pode comparar grandezas entre Strings

Operadores Lógicos: símbolos especiais que realizam comparações lógicas entre operandos lógicos ou expressões
- Conjunções: &&/and
- Disjunção: ||/or
- Disjunção Exclusiva: ^/xor (operandos opostos ex: V F = V)
- Negação: !/inverção

Controle de Fluxo: direcionam o fluxo de execução do código
- Decisão: if, switch, operador ternário (ex: ligado? desligar:ligar;)
- Repetição: for, while, do while
- Interrupção: break, continue, return

Blocos: conjunto de códigos que trabalham juntos para executar uma operação {}
- Locais: dentro de métodos
- Estáticos: dentro de classes
